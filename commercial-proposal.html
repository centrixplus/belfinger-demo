Features

Complete Scope of Deliverables

  • ✓
    7-Step Approval Workflow Engine — Full cycle: MEW Request → Manager Review → Supervisor Assignment → Engineer Work → Work Report → Invoice → MEW Approval, with rejection & rework at every stage
  • ✓
    Work Order Management — Issue Requests (MEW-initiated) + Preventive Maintenance (auto-scheduled), SLA tracking with urgency levels, real-time status updates 24/7
  • ✓
    Complete BOQ Catalog & Data Entry — All 300+ BOQ items across 7 pricing tables entered, categorized by department, with unit prices and versioning
  • ✓
    Full Equipment Register — 37 Sites — Every piece of equipment across 20 pump stations & 17 water towers: tag numbers, categories, technical specs, maintenance history, status (working/defective)
  • ✓
    Invoice Management & Receivables — BOQ-driven invoice drafting, multi-level approval chain, monthly receivables tracking, rejection with line-item comments
  • ✓
    Dashboard & Real-Time KPIs — Jobs by department, status breakdown, SLA compliance, invoice pipeline, monthly receivables chart, PM schedule overview
  • ✓
    Equipment Checklists & Calibration — Per-equipment-type inspection checklists, calibration matrices with auto-calculated % error (0/25/50/75/100%)
  • ✓
    Digital Signatures & Photo Evidence — Dual digital signatures (Belfinger + MEW) with GPS coordinates, photo upload with annotation
  • ✓
    Historical Data Migration — Import all existing records from Excel files: equipment lists, maintenance tables, BOQ data, executed work reports (invoices)
  • ✓
    Multi-Device Web Access — Responsive design works on PC, laptop, tablet, and mobile — accessible globally via any web browser, 24/7
  • ✓
    Bilingual EN/AR with RTL — Full Arabic UI with right-to-left layout, all content fields support bilingual entry
  • ✓
    User Roles & Permissions — 6 role types (MEW, Belfinger Manager, Contractor Manager, Supervisor, Site Engineer, Admin) with granular access control
  • ✓
    Notifications & Alerts — In-app + email notifications for assignments, approvals, rejections, SLA warnings, and escalations
  • ✓
    PDF Reports & Exports — Monthly work reports, maintenance summaries, SLA compliance reports, invoice PDFs — matching MEW submission format
  • ✓
    Audit Trail & Compliance — Tamper-evident activity log: who did what, when, with full traceability for MEW audits
  • ✓
    Server Setup, Security & Deployment — Production server configuration, SSL encryption, automated backups, RBAC security
  • ✓
    On-Site Training & Documentation — Hands-on training sessions for every user role, admin guide, user manual
